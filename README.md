# Vol-Platform Frontend

## Опис
Це фронтенд частина платформи для замовлення клінінгових послуг. Реалізовано на React + Vite, з інтеграцією Firebase (Firestore, Auth) та REST API для рейтингу/замовлень.

## Основні можливості
- Динамічний список послуг з Firestore
- Оформлення замовлення з вибором площі, регулярності, періоду
- Динамічний розрахунок ціни та знижок
- Авторизація користувачів (Firebase Auth)
- Адмін-панель для перегляду та зміни статусу замовлень
- Календар замовлень для адміністратора
- Відгуки та рейтинг послуг

## Структура проекту
- `src/pages` — сторінки (послуги, замовлення, історія, адмін, календар)
- `src/components` — компоненти (ServiceCard, StarRating, NavMenu тощо)
- `src/firebase` — налаштування та сервіси Firebase

## Запуск
1. Встановіть залежності:
   ```sh
   npm install
   ```
2. Створіть файл `.env` у корені `vol-platf`:
   ```env
   VITE_API_BASE_URL=http://localhost:5000
   ```
3. Запустіть фронтенд:
   ```sh
   npm run dev
   ```
4. Відкрийте [http://localhost:5173](http://localhost:5173)

## Важливо
- Для роботи потрібен запущений бекенд (див. інструкцію у папці `ServerWebLab5-main`).
- Для повної функціональності потрібні налаштовані Firebase Auth та Firestore (див. `FIREBASE_SETUP.md`).
- Адмін-панель та календар доступні лише користувачам з роллю `admin` у Firestore.
